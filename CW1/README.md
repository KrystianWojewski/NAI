# NAI

### Jaka metoda daje najlepsze wyniki?
###### Najlepsze wyniki dla najmniejszej ilości iteracji daje metoda pełnego przeglądu (tabu). Jej wynik jest najbliższy poprawnemu wynikowi czego nie można powiedzieć o pozostałych funkcjach natomiast czas działania algorytmu pozostawia wiele do życzenia. Metoda tabu działa najdłużej z innych algorytmów

### Jaka metoda jest najszybsza dla podobnej jakości wyników? Jaka metoda kompletnie się nie nadaje?
###### Algorytm wspinaczkowy jest najszybszy ze wszystkich trzech algorytmów pod względem tej samej ilości iteracji. Nie zawszę daję najdokładniejszy wynik lecz przy większej ilości iteracji algorytm wspinaczkowy jest dokładnieszy.
###### Algorytm symulowanego wyżarzania jest pomiędzy algorytmem wspinaczkowym a metodą tabu. Algorytm symulowanego wyżarzania jest szybszy od metody tabu ale za to mniej dokłady oraz jest wolniejszy od algorytmu wspinaczkowego, ale dokładniejszy.
